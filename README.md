# Lead-Scout CLI 
**Autonomous GTM Intelligence Agent**

Lead-Scout is a Python-based Command Line Interface (CLI) tool designed to automate deep-prospect research. Developed using **Claude Code** and **Agentic Design Patterns**, it bypasses the surface-level data of standard GTM tools to provide a "Hypothesis of Need" based on real-time triggers.

##  Tech Stack
* **Language:** Python 3.11+
* **AI Orchestration:** Anthropic Claude API / OpenAI SDK
* **Development Environment:** Claude Code CLI, MCP (Model Context Protocol)
* **Libraries:** Typer (CLI), Rich (Terminal Formatting), Dotenv (Security)

##  Key Features
* **Trigger Event Identification:** Scans for expansion, funding, or leadership changes.
* **Contextual Outreach Generation:** Generates hyper-personalized scripts based on technical "pain-point" hypotheses.
* **CRM-Ready Output:** Clean JSON data ready for HubSpot/Salesforce ingestion.

##  Technical Mastery
This project demonstrates proficiency in:
1. **Agentic Workflows:** Moving beyond prompts to autonomous scripts.
2. **CLI Development:** Building tools for technical sales operations.
3. **Secure API Management:** Implementing environment variables to protect enterprise credentials.
